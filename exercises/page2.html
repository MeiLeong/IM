This is the anatomy of a Pet

We will start with the Neck.
A microchip is inserted between the base of the neck and the top of the shoulder blades.
The neck can be scanned using a handheld scanner (network) to retrieve the data from the chip.

Microchipping allows humans to store information about their pets, within the animal’s own body. The Microchip has an individual identifier, which can be used to lookup the owner’s information in a Pet registry database.
